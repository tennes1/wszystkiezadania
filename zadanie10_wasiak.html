1. Części selektora:

Selektor – wskazuje, do jakiego elementu HTML mają być zastosowane style
Deklaracja – zawiera zestaw cech i wartości stylu

2. Znaczenie nawiasów klamrowych { ... }

Nawiasy klamrowe otaczają deklarację stylu. Wewnątrz nich zapisujemy cechy i wartości.

3. Opis cech:

Cechy (np. color, font-size) to właściwości, które zmieniają wygląd elementu.

4. Opis wartości:

Wartości (np. red, 20px) to konkretne ustawienia dla cech.

5. Kiedy stosujemy średnik:

Średnik oddziela każdą parę cecha: wartość. Ostatni średnik można pominąć, ale dla porządku lepiej go dodać.

Przykład poprawnego selektora:

p {
  color: blue;
  font-size: 16px;
}
